## Quick install

1. Install [Pixi](https://pixi.sh/), a (modern) package management tool

2. Get a clone of this repository:

```
$ git clone https://github.com/Alex-bzh/python-M1LTE.git
$ cd python-M1LTE
```

3. Install the dependencies:

```
$ pixi install
```
